¿Qué es HTML5?

HTML5 es un lenguaje de marcas que nos permite el desarrollo de páginas web, al cual se le han añadido un conjunto de capacidades que nos permiten explotar las capacidades de los dispositivos en temas de multimedia, rendimiento, offline, interacción con el usuario,… De esta manera HTML5 se compone de un núcleo de elementos del lenguaje y una serie de APIs que nos permiten gestionar las nuevas capacidades.

¿Qué es CSS3?

CSS es uno de los lenguajes base de la Open Web y posee una especificación estandarizada por parte del W3C. Anteriormente , el desarrollo de varias partes de las especificaciones de CSS era realizado de manera sincrónica, lo que permiía el versionado de las recomendaciones. Probablemente habrás escuchado acerca de CSS1, CSS2.1, CSS3. Sin embargo, CSS4 nunca se ha lanzado como una versión oficial.
